This page generates a new Monero wallet. It is self contained and does all the necessary calculations locally, so is suitable for generating a new wallet on a machine that is not connected to the network, and may even never be. This way, you can create a Monero wallet without risking the keys. This file is GPG signed, see GPG instructions. You can check for up to date versions of this page here.

Custom entropy for deterministic wallet (leave empty to use the browser's PRNG)
